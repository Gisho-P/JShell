User Stories from Product Backlog that we will address in this sprint:


Specific tasks to be accomplished:
a) As a user, I can create a directory that I can go through and create files in.
b) As a user, I can traverse through directories to look for/create/delete files, etc.
c) As a user, I can keep track of what directory I'm in by displaying it's path.
d) As a user, I can choose a file and look at it's contents.
e) As a user, I can look at the history of commands I entered in the shell.
f) As a user, I can input commands to the shell to perform different tasks. 
g) As a user, I can display the manual to a command I don't understand.

These tasks are assigned as follows:
a-1. (Directory) Create method to add an object to the directory
b-1. (Directory) Create methods to get parent and child objects of current directory
b-2. (FilePathInterpreter) Create a class that interprets a path, parses it and goes to the location
c-1. (Directory) Create a method to return path
d-1. (FileTypes) Create a new superclass that created files/directories inherit from
d-2. (File) Make File inherit from FileTypes
e-1. (Session) Display history using Iterator structure
f-1. (Command) Create an abstract super class command for all commands entered
g-1. Write man pages for each command
g-2. (CommandInterpreter) Implement method in class to get man pages and display them
