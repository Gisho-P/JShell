NAME
        ls - prints out all of the contents of one or many files/directories

SYNOPSIS
        ls [PATH ...]

DESCRIPTION
        Prints out the contents of files/directories.

        If PATH is not specified, prints out the contents of the current
        directory by default.

        If PATH is specified, prints out the contents of the files/directories
        for each PATH given
