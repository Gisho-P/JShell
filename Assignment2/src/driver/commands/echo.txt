ECHO(1)                          User Commands                          ECHO(1)

NAME
        echo - prints a string to standard output, or saves a string to a text
        file

SYNOPSIS
        echo STRING [> OUTFILE]

DESCRIPTION
        Saves the STRING to the OUTFILE if it's provided, otherwise it prints
        the STRING to the standard output.

        If the OUTFILE is an existing file in the directory, it will overwrite
        the contents of the OUTFILE with the string, otherwise it will create
        a new file with name OUTFILE .

        STRING should be a string of characters surrounded by double quotation
        marks.
