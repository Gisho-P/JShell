Class name: Directory
Parent class (if any): FileTypes
Subclasses (if any): None

Responsibilities:
* Stores the file objects
* Stores the name of the current directory
* Stores parent and all child dirs
* Checks if file or dirs exists within it
* Allows file addition, deletion and search
* Allows traversal to parent and child directories from current directory
* Determine the current path from current directory to the root directory
* Return information of files and directories in current directory 

Collaborators:
* File
* InvalidNameException
* InvalidAdditionException
* MissingNameException
